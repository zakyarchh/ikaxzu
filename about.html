FAQ

Bagaimana cara menonton live?

Beli token atau membership di Shop, lalu masuk ke room live yang tersedia. Gunakan token untuk mengakses streaming.

Apa bedanya token dan membership?

Token untuk sekali nonton (1 token = 1 akses), membership untuk akses unlimited selama periode aktif (1-12 bulan).

Kenapa tidak bisa login?

Token hanya bisa digunakan di 1 perangkat. Jika sudah digunakan di perangkat lain, silahkan hubungi admin untuk reset.
